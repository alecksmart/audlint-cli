# Contributing

Thanks for your interest in `audlint-cli`.

## What we welcome
- Bug reports
- Feature requests

Please open a GitHub Issue with:
- what you expected,
- what happened,
- clear reproduction steps,
- environment details (`OS`, shell, major tool versions).

## Pull requests
At this stage, development is maintainer-led. PRs may be accepted selectively.
Open an issue first before implementing significant changes.

## Scope
The project focuses on music library quality browser and related CLI workflows.
Requests outside this scope may be declined.
